Formulario

Los formularios se incluyen dentro de la etiqueta form. En ellos se utlizan los siguientes controles (elementos del formulario que son activos; con los que el usuario puede interactuar).

  • Input text
  • Text ares
  • Checkbox
  • Radio buttons
  • Select / options
  • Botones

La etiqueta form tiene dos atributos importantes

  • Action
  • Method
